## 1. 진화 컨셉

**테마: "직장인의 성장"** — 신입의 고충이 연차가 쌓이며 자기만의 스타일로 완성된다.
진화해도 귀여움은 유지하고, **기존 실루엣에서 알아볼 수 있게** 변화는 소품·색·디테일 중심.

| 규칙 | 내용 |
|------|------|
| 시점 | 부화 후 7일 (성체 진입 순간, 변신 연출과 함께) |
| 방향 | 캐릭터당 1단계 진화 (분기 없음 — Plan FR-15 v2) |
| 크기 | 기존 성체와 동일 스케일 (게임에서 동일 캔버스 256px) |
| 정체성 | 색 팔레트·얼굴은 유지, 소품과 자세로 "성장"을 표현 |

## 2. ⚠️ GPT 요청 시 핵심 팁

1. **기존 캐릭터 이미지를 반드시 첨부**하고 "이 캐릭터의 진화형"으로 요청 — 얼굴·스타일 일관성의 핵심!
2. 스타일·기술 조건은 [chatgpt-image-prompts.md](chatgpt-image-prompts.md)의 **공통 스타일 블록**을 먼저 붙여넣기 (치비·반짝이 눈·두꺼운 외곽선·투명 PNG)
3. **4열×2행 포즈 시트** 동일 구성: 기본 / 걷기A / 걷기B / 잠 / 신남 / 시무룩 / 아픔 / 먹기
4. 완성 파일은 `design/` 폴더에 `진화_캐릭터이름.png`로 저장 (예: `진화_모찌.png`)

## 3. 진화형 10종

### 3.1 모찌 🍡 → 프로찌 (일잘러 슬라임)

| 항목 | 내용 |
|------|------|
| 스토리 | 매일 흘러내리던 신입이 어느새 팀의 기둥. 여전히 말랑하지만 흔들리지 않는다 |
| 외형 변화 | 몸이 한층 탱탱하고 광택이 매끈해짐. **작고 동그란 안경** + 목에 조그만 **파란 넥타이**. 흘러내리는 디테일은 여유로운 미소와 함께 "일부러 흘러내리는" 느낌 |
| 성격 변화 | 수줍음 → 차분한 자신감. 칭찬을 받아도 이제 덜 녹는다 |

```
(모찌 이미지 첨부 후) 이 캐릭터의 진화형 "프로찌"의 포즈 시트를 만들어줘:
같은 분홍 슬라임인데 2년차 일잘러가 된 모습. 작고 동그란 안경을 쓰고
목에 조그만 파란 넥타이. 광택이 더 매끈하고 자신감 있는 차분한 미소.
얼굴과 색감은 원본과 똑같이 유지해줘.
```

목의 사원증은 금색 스트랩으로 업그레이드, 한쪽 날개에 테이크아웃 커피컵.
