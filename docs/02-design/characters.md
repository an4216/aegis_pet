## 1. 디자인 공통 원칙

### 1.1 픽셀아트 스펙

| 항목 | 기준 |
|------|------|
| 캔버스 | 유아기 32×32 / 소년기 40×40 / 성체 48×48 px |
| 스타일 | 2등신 블롭형, 두꺼운 다크톤 외곽선(1px), 도트 감성 |
| 팔레트 | 캐릭터당 **메인 2색 + 명암 1색 + 포인트 1색** (총 4색 이내) |
| 실루엣 원칙 | 색을 빼고 실루엣만 봐도 10종이 구분되어야 함 (귀/볏/꼬리/소품으로 차별화) |
| 애니메이션 기본 세트 | Idle 4f / Walk 6f / Sleep 4f / Eat 4f / Drag 2f / **시그니처 Idle 4~6f** |
| 성장 표현 | 유아기는 전 종족 공통 실루엣에 색+포인트만 반영 → 소년기부터 종족 실루엣 발현 |

### 1.2 캐릭터 컨셉 원칙

- **직장인 페르소나**: 10종 모두 사무실에서 볼 법한 인간 군상을 동물/사물에 투영
- **어깨너머 안전 원칙**: 말풍선은 상사가 어깨너머로 봐도 웃고 넘어갈 수준까지만 (회사·특정인 직접 비난 금지, 자조는 가볍게)
- **위로 > 냉소**: 뼈 때리는 대사 뒤에는 반드시 다독임 계열 대사가 섞이도록 대사 풀 구성

## 2. 부화 시스템

### 2.1 부화 확률

| 등급 | 캐릭터 | 기본 확률 |
|------|--------|----------|
| 일반 (60%) | 모찌, 삐약, 햄찌, 꾸벅 | 각 15% |
| 희귀 (32%) | 나른냥, 콩이, 문덕, 거부장 | 각 8% |
| 레어 (8%) | 불금조, 스르륵 | 각 4% |

### 2.2 히든 가중치 (알 시절 행동이 부화에 영향)

| 조건 | 가중치 효과 |
|------|-------------|
| 알을 자주 클릭/쓰다듬음 (상위 케어) | 삐약 ×2 |
| 밤 시간(22시~)에 부화 | 스르륵 ×3 |
| 금요일에 부화 | 불금조 ×3 |
| 점심시간(12~13시)에 부화 | 햄찌 ×2 |
| 거의 방치 상태로 부화 | 나른냥 ×2, 거부장 ×2 |
| 오전 7~9시(출근시간)에 부화 | 콩이 ×2 |

> 가중치는 기본 확률에 곱연산 후 전체 재정규화. 도감(Phase 3)에서 힌트로 노출.

## 4. 말풍선 시스템

### 4.1 표시 규칙

| 항목 | 규칙 |
|------|------|
| 빈도 | 기본 1~3분 랜덤 (심심하면 계속 조잘). 설정: 자주 30~90초 / 보통 1~3분 / 가끔 20~40분 / 끄기 |
| 표시 시간 | 5초 후 자동 소멸, 클릭 시 즉시 닫힘 |
| 동시 표시 | 항상 1개만 |
| 집중 모드 | 전체 비활성 |
| 위치 | 펫 머리 위, 화면 밖으로 나가지 않게 클램프 |
| 대사 선택 | 캐릭터 전용 풀 60% + 공통 풀 40%, 최근 5개 중복 금지 |
| 톤 가이드 | 어깨너머 안전 원칙(§1.2) 준수. 위로:유머:자조 = 4:4:2 비율 유지 |
