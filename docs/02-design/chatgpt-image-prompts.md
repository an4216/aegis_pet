# ChatGPT 캐릭터 이미지 생성 가이드 (포즈 시트 방식)

> 목적: 캐릭터 11종(알 포함)을 고퀄 일러스트 + 걷기·표정 애니메이션용 포즈 시트로 재제작 → 게임 적용
> 캐릭터 설정 원본: [characters.md](characters.md)

---

## 📐 왜 "포즈 시트" 1장으로 받나?

AI는 **같은 이미지 안에서는** 캐릭터를 일관되게 그리지만, 이미지를 나눠 생성하면 매번 얼굴이 달라집니다.
그래서 캐릭터당 **8칸 격자 시트 1장**으로 받고, 제가 칸별로 잘라 게임 애니메이션 프레임으로 씁니다.

**8가지 표준 포즈** (게임 상태와 1:1 대응):

| 칸 | 포즈 | 게임에서 쓰이는 곳 |
|:--:|------|-------------------|
| 1 | 기본 서있기 (미소) | 대기 |
| 2 | 걷기 A (몸 살짝 기울고 한쪽 발 앞) | 걷기 (3번과 번갈아 재생) |
| 3 | 걷기 B (반대쪽 기울기·발) | 걷기 |
| 4 | 자는 얼굴 (눈 감고 편안) | 수면·집중 모드 |
| 5 | 신남 (활짝 웃고 팔 번쩍/점프) | 놀기·축하·쓰다듬기 |
| 6 | 시무룩 (처진 눈썹, 웅크림) | 행복 낮을 때 |
| 7 | 아픔 (@_@ 어지러운 눈, 창백) | 병들었을 때 |
| 8 | 먹기 (볼 빵빵, 오물오물) | 먹이/간식 |

## ⚠️ 반드시 지켜야 할 기술 조건 (이걸 안 지키면 적용 불가/재작업)

1. **배경 투명 PNG** — "배경을 진짜 투명하게(alpha transparency). 체커보드 무늬를 그려넣지 말 것"이라고 명시
2. **4열 × 2행 균등 격자**, 각 칸에 같은 캐릭터 1마리씩, 칸 중앙 배치, **모든 칸에서 캐릭터 크기 동일**
3. 캐릭터끼리 겹치지 않게, 칸 사이 여백 확보
4. **글자·텍스트·번호·칸 테두리선 없음** (제가 위치로 구분합니다)
5. 크기 1024×1024 이상 (칸당 256px 확보)
6. **스타일 통일**: 한 대화방에서 이어서 생성, 두 번째 캐릭터부터는 "위와 완전히 같은 스타일로"

## 💡 진행 팁

- 첫 캐릭터(모찌)로 스타일이 마음에 들 때까지 반복 → 확정되면 나머지를 같은 방에서 이어서
- 마음에 안 들면 "눈을 더 크게", "외곽선 더 두껍게"처럼 부분 수정 요청
- 완성본은 아무 폴더에나 저장 (예: `바탕화면\새캐릭터\`) — 파일명은 아무거나, 어떤 캐릭터인지만 알려주시면 제가 매핑·적용합니다

## 0. 공통 스타일 (첫 메시지에 먼저 붙여넣기)

```
지금부터 데스크톱 펫 게임용 캐릭터 포즈 시트를 만들 거야. 캐릭터마다 이미지 1장을 만들 건데, 반드시 아래 규칙을 따라줘:

[스타일]
- 카카오톡 이모티콘 같은 귀여운 치비(chibi) 스타일, 2등신
- 크고 반짝이는 눈 (큰 하이라이트 + 작은 하이라이트 2개)
- 두껍고 부드러운 다크브라운 외곽선 (스티커 느낌)
- 파스텔톤 플랫 컬러 + 최소한의 부드러운 음영, 볼터치 필수

[포즈 시트 구성 — 4열 × 2행 격자, 왼쪽 위부터 순서대로]
1. 기본 서있기 (정면, 미소)
2. 걷기 A (몸을 왼쪽으로 살짝 기울이고 한쪽 발/몸을 앞으로)
3. 걷기 B (2번과 반대 방향 기울기)
4. 자는 모습 (눈 감고 편안한 얼굴)
5. 신나는 모습 (활짝 웃으며 폴짝 뛰는 포즈)
6. 시무룩한 모습 (처진 눈썹, 살짝 웅크림)
7. 아픈 모습 (어지러운 @_@ 눈, 힘없는 표정)
8. 먹는 모습 (볼이 빵빵, 오물오물)

[기술 조건]
- 8칸 모두 완전히 같은 캐릭터, 같은 크기, 각 칸 중앙 배치
- 칸 테두리선·번호·글자·워터마크 절대 넣지 마
- 배경은 진짜 투명한 PNG (체커보드 무늬를 그리지 말고 alpha 투명으로)
- 1024×1024 이상
```

## 1~11. 캐릭터별 프롬프트 (한 개씩 순서대로)

### 1. 모찌 🍡
```
첫 번째 캐릭터 "모찌"의 포즈 시트를 만들어줘: 연분홍색 찹쌀떡 슬라임. 팔다리 없는 매끈한 반달(물방울) 모양 젤리 몸. 몸 한쪽이 살짝 흘러내리는 디테일. 머리에 젤리 광택 하이라이트. 순하고 수줍은 성격.
메인 컬러: #FFD9E8, 볼터치: #FF8FB1
(팔다리가 없으니 걷기 포즈는 몸을 좌우로 기울여 통통 튀는 느낌으로)
```

### 2. 삐약 🐤
```
같은 스타일로 "삐약": 동그란 노란 병아리. 머리 위에 새싹처럼 삐친 깃털 한 가닥. 목에 파란 스트랩의 회사 사원증을 걸고 있음(사원증에 글자는 넣지 말 것). 초롱초롱 열정 가득한 눈. 짧은 날개를 파닥이는 포즈.
메인 컬러: #FFE066, 부리·발: #FF9F1C
```

### 3. 햄찌 🐹
```
같은 스타일로 "햄찌": 크림색 햄스터, 머리에 주황색 얼룩 하나. 볼주머니가 빵빵하게 부풀어 있음. 두 손으로 해바라기씨를 들고 오물오물 먹는 중. 배는 밝은 크림색.
메인 컬러: #FFE8C8, 얼룩: #F4A259
```

### 4. 꾸벅 🐻
```
같은 스타일로 "꾸벅": 연갈색 아기 곰. 반쯤 감긴 졸린 눈, 눈 밑에 옅은 보라색 다크서클. 목에 하늘색 담요를 목도리처럼 두르고 있음. 입은 작게 하품하거나 편안한 표정.
메인 컬러: #C8A180, 담요: #BFD3E6
```

### 5. 나른냥 🐈
```
같은 스타일로 "나른냥": 회색 고양이. 반쯤 감긴 나른한 눈, 축 처진 꼬리. 입에 흰색 편지봉투를 물고 있음(봉투에 빨간 동그란 씰 하나, 글자는 없음). 만사 귀찮은 표정.
메인 컬러: #9AA0A6, 봉투: 흰색
```

### 6. 콩이 ☕
```
같은 스타일로 "콩이": 갈색 커피콩 캐릭터. 콩 가운데 세로 크랙 라인. 머리 위로 김이 모락모락 두 줄기. 카페인 각성으로 크게 뜬 눈(흰자 넓고 동공 작음), 놀란 듯한 'o' 입.
메인 컬러: #8B5E3C, 김: 크림색
```

### 7. 문덕 🐙
```
같은 스타일로 "문덕": 연보라색 아기 문어. 물결치는 다리 8개 중 한 다리엔 서류 뭉치, 다른 다리엔 커피컵을 들고 있음(서류에 글자 없음). 걱정스러운 팔자 눈썹 + 작은 'o' 입 + 이마에 땀방울 하나.
메인 컬러: #B39DDB
```

### 8. 거부장 🐢
```
같은 스타일로 "거부장": 진녹색 거북이. 등껍질이 서류가방처럼 생김(가방 손잡이가 위로 솟음, 가운데 금색 잠금쇠). 코끝에 걸친 동그란 노안 안경, 인자한 미소. 나이 지긋한 부장님 느낌.
메인 컬러: #6B8E23, 안경·손잡이: 카키 골드
```

### 9. 불금조 🔥
```
같은 스타일로 "불금조": 주황-빨강 불사조 새. 머리에 불꽃 모양 볏이 활활 타오름. 배는 노란색. 신나서 눈을 반짝이는 표정. 꽁지깃도 작은 불꽃 모양.
메인 컬러: #FF6B35, 볏·배: #FFD166
```

### 10. 스르륵 👻
```
같은 스타일로 "스르륵": 연하늘색 반투명 유령. 몸 아래가 연기처럼 세 갈래로 갈라짐. 목에 흰 셔츠 카라 + 빨간 넥타이(작게). 여유로운 미소. 살짝 투명한 느낌이 나면 좋음.
메인 컬러: #A8DADC, 넥타이: #E63946
```

### 11. 알 🥚
```
같은 스타일로 "알": 크림 아이보리색 알. 분홍색 얼룩무늬 3개. 윗부분에 반질반질한 광택 하이라이트. 얼굴은 없음.
메인 컬러: #FFF8E6, 무늬: #FFB4C8
(알은 포즈 시트 대신 4칸만: 1.가만히 / 2.왼쪽으로 갸우뚱 / 3.오른쪽으로 갸우뚱 / 4.금이 살짝 간 모습 — 2열×2행)
```

> 캐릭터별 프롬프트(2~10번)는 기존 외형 설명 그대로 쓰되, 첫 문장을 "같은 스타일로 ○○의 포즈 시트를 만들어줘"로 시작하면 됩니다.

## 받은 뒤 제가 하는 작업

1. 투명배경 검수 (불투명이면 배경 제거 시도)
2. 격자 8칸 슬라이스 → 포즈별 프레임 추출·정렬·트리밍
3. 게임 애니메이션 시스템 업그레이드: 걷기 2프레임 교차 재생, 상태별(잠·신남·시무룩·아픔·먹기) 표정 전환
4. 파일 배치 → 게임 내 크기·전환 확인 → 새 버전 릴리스 (기존 사용자 자동 업데이트)
